# cluster-ctrl

This repo is reserved for the **cluster-ctrl Operations Console** — the web application for
Day-2 administration of the hardened Google Kubernetes Engine (GKE) clusters (see OPS-1..5 in
the requirements). Console work has not started yet.

The platform documentation that used to live here — requirements, technology choices,
security requirements, the Google Cloud design, and the Architecture Decision Records — moved
(with full git history) into the implementation repo on 2026-06-09, so design and
implementation evolve together:

**→ [`iac-gke/docs/`](https://github.com/kg-aifabrik/iac-gke/tree/main/docs)**
